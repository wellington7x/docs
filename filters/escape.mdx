### Introdução

O filtro `escape` é usado para converter caracteres especiais em uma string para suas entidades de texto equivalentes. Isso é útil para garantir que o conteúdo exibido em mensagens de texto seja seguro e corretamente formatado sem interpretar caracteres especiais.

### Exemplo de uso

Aqui está um exemplo de como usar o filtro `escape` para garantir que caracteres especiais em uma string sejam exibidos corretamente:

```twig
{{ 'Olá, essa é uma mensagem teste'|escape }}
```

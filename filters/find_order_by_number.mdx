### Introdução

O filtro `find_orders_by_email` é um método utilizado para buscar pedidos realizados conforme o número do pedido informado pelo cliente. Ele é útil para criar consultas de pedidos em automações de chatbot, por exemplo. Sendo possível exibir informações sobre o pedido. Lembrando que nesse caso, o sistema irá apenas o pedido associado ao número de pedido informado pelo cliente.

### Exemplo de uso

Exemplo de como utilizar o filtro `find_order_by_number`:

```twig
{% set orders = find_order_by_number(number) %}

{% if orders is defined and orders is not empty %}
  Encontramos pedido associado ao número: {{ number }}.
{% else %}
  Não há pedido associado ao número: {{ number }}.
{% endif %}
```

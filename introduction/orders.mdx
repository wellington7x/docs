---
title: "Dados de Pedidos"
description: "*Variáveis disponíveis para imprimir dados de pedidos em mensagens.*"
---

Estão disponíveis para uso somente em mensagens destinadas a pedidos. Para utilizar dados de pedidos em suas mensagens, você _deve categorizar a mensagem_ com o **Tipo PEDIDO**. Ao fazer isso, nosso sistema realizará a leitura dos dados de pedidos e permitirá seu uso adequado.

#### Dados Gerais do Pedido

| **Nome**                                     | **Variável**                                    | **Saída**                                                                 | **Tipo** |
| -------------------------------------------- | ----------------------------------------------- | ------------------------------------------------------------------------- | -------- |
| Data de Venc. do Boleto                      | `{{ order.billet_expired_at_formatted }}`       | 12/02/2024                                                                | String   |
| Linha digitável do boleto / Pix Copia e Cola | `{{ order.billet_line }}`                       | Código de Pagamento                                                       | String   |
| Link do boleto/pagamento                     | `{{ order.billet_url }}`                        | link.pagamento/exemPLO                                                    | String   |
| Moeda                                        | `{{ order.currency }}`                          | BRL                                                                       | String   |
| Número do pedido                             | `{{ order.number }}`                            | João Teste                                                                | String   |
| Valor subtotal                               | `{{ order.subtotal_price }}`                    | 9.90                                                                      | Number   |
| Valor subtotal (Formatado)                   | `{{ order.subtotal_price_formatted }}`          | R$9,90                                                                    | String   |
| Valor total                                  | `{{ order.total_price }}`                       | 210.27                                                                    | Number   |
| Valor total (Formatado)                      | `{{ order.total_price_formatted }}`             | R$210,27                                                                  | String   |
| Endereço completo (Formatado)                | `{{ order.shipping_address_formatted }}`        | Rua Flores, 456 - C - Bairro dos Sonhos, Florianópolis, SC, 06970-000, BR | String   |
| Produtos (Formatado)                         | `{{ order.line_items_formatted }}`              | Produto 01 - AB                                                           | Array    |
| Produtos com Variante (Formatado)            | `{{ order.line_items_formatted_with_variant }}` | Produto 01 - AB - Variante / C                                            | String   |

#### Itens do Pedido (order.line_items) - Array

| **Nome**                      | **Variável**                    | **Saída**                      | **Tipo** |
| ----------------------------- | ------------------------------- | ------------------------------ | -------- |
| Itens Pedido (Array Completo) | `{{ order.line_items }}`        | `[{"title": "Meu Robô"}, ...]` | Array    |
| Título                        | `{{ line_item.title }}`         | Meu Robô                       | String   |
| Nome da Variante              | `{{ line_item.variant_title }}` | Azul                           | String   |
| Quantidade                    | `{{ line_item.quantity }}`      | 1                              | Number   |
| Preço                         | `{{ line_item.price }}`         | 210.27                         | Number   |
| URL da Imagem                 | `{{ line_item.image_url }}`     | https://….png                  | String   |

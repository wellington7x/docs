---
title: "Dados de Rastreios"
description: "*Variáveis disponíveis para imprimir dados de rastreios em mensagens.*"
---

Estão disponíveis para uso somente em mensagens destinadas a pedidos que possuem um número de rastreio atrelado. Utilizado para exibir informações sobre o rastreio do cliente. Lembrando que nesse caso as variáveis irão funcionar somente se houver um pedido criado com um número de rastreio atrelado.

#### Dados Gerais de Rastreio

| **Nome**           | **Variável**                       | **Saída**                | **Tipo** |
| ------------------ | ---------------------------------- | ------------------------ | -------- |
| Código de rastreio | `{{ tracking_number.code }}`       | NM381645789BR            | String   |
| Link de rastreio   | `{{ order.tracking_numbers_url }}` | rastreio.pl/pag.rastreio | String   |

#### Eventos de Rastreio (tracking_number.events) - Array

| Nome        | Variável                            | Saída                                           | Tipo   |
| ----------- | ----------------------------------- | ----------------------------------------------- | ------ |
| Eventos     | `{{ tracking_number.events }}`      | `[{"type": "Correios"}, ...]`                   | Array  |
| Tipo        | `{{ tracking_number.type }}`        | CORREIOS                                        | String |
| Data        | `{{ tracking_number.date }}`        | 2024-02-06 08:32:48                             | String |
| Rótulo      | `{{ tracking_number.label }}`       | Objeto em transferência - por favor aguarde     | String |
| Localização | `{{ tracking_number.location }}`    | Unidade de Tratamento - SAO JOSE DOS CAMPOS, SP | String |
| Descrição   | `{{ tracking_number.description }}` | Aguardando mais informações                     | String |
